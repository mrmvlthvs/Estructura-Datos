# Security Policy

## Supported Versions

This repository contains academic projects developed for the Data Structures course.
Only the latest version available in the main branch is currently maintained.

| Version | Supported          |
| ------- | ------------------ |
| Latest (main)  | :white_check_mark: |
| Older versions   | :x:                |


## Reporting a Vulnerability

This project is intended for educational and academic purposes.
If you discover a bug, vulnerability, unexpected behavior, or implementation issue, please open a GitHub Issue and provide:

- A clear description of the problem.
- Steps to reproduce the issue.
- The affected file or module.
- A possible solution (optional).

Reports will be reviewed when possible; however, there is no guarantee of continuous maintenance or response times.

Thank you for helping improve this project.
