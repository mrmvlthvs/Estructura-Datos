# Estructura de Datos

## Descripción
Este proyecto contiene una colección de ejercicios de programación en 
**Java** desarrollados, compilados y codificados para la materia de 
**Estructura de Datos**. 
En este proyecto se encontrara implementaciones prácticas de conceptos 
fundamentales como:

- **Listas** (enlazadas y dinámicas)
- **Pilas** (Stacks)
- **Colas** (Queues)
- **Árboles** (binarios, de búsqueda)
- **Grafos**
- **Tablas Hash**
- **Ordenamiento y Búsqueda**

## Objetivo

Reforzar los conocimientos teóricos mediante código funcional y ejercicios 
progresivos que van desde lo básico hasta casos más complejos.

## Estructura

Cada ejercicio está organizado en paquetes temáticos con comentarios 
explicativos del código.
