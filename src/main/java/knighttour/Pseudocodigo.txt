3.1. isSafe(x, y)
    retornar (x >= 0 Y x < N Y y >= 0 Y y < N Y sol[x][y] es igual a -1)

3.2. solveKTUtil(x, y, movei)
    Entero: k, next_x, next_y

    // Caso base: Si todos los movimientos han sido completados
    si (movei es igual a N * N) entonces empezar
        retornar verdadero
    terminar
    
    // Evaluar los 8 movimientos posibles del caballo
    desde k = 0; mientras k < N; incrementar k para empezar
        next_x = x + xMove[k]
        next_y = y + yMove[k]
        
        si (llamar isSafe(next_x, next_y)) entonces empezar
            sol[next_x][next_y] = movei
            
            si (llamar solveKTUtil(next_x, next_y, movei + 1)) entonces empezar
                retornar verdadero
            terminar
            sino empezar
                sol[next_x][next_y] = -1 // Vuelta Atrás (Backtracking)
            terminar
        terminar
    terminar
    
    retornar falso
